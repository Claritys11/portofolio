# **App Name**: Code Chronicles

## Core Features:

- Article Listing: Display a list of articles on the homepage, including title, category, date, and estimated reading time.
- Category and Tag Filtering: Allow users to filter articles based on categories (Write-Up, Blog) and tags (National, SHS, International).
- Full Article Display: Display the full article content, parsed from Markdown, with metadata (date, reading time, category, tags).
- Sidebar Navigation: Implement a sidebar for easy navigation to Home, About, Archives, Search, and external Links.
- Full-Text Search: Enable users to search for articles using keywords.
- Archive Browsing: Allow users to browse articles by month and year in the archive section.
- Content Generation Suggestion: Suggest potential new content ideas, using current content, browsing trends and external content using a LLM as a tool.

## Style Guidelines:

- Primary color: Dark blue (#2c3e50) for a professional and readable feel.
- Background color: Light gray (#f0f2f5) to ensure readability and a clean design.
- Accent color: Teal (#3498db) to highlight interactive elements.
- Body font: 'PT Sans', a sans-serif for readability; headlines will use 'Space Grotesk'.
- Use minimalist icons for categories, tags, and navigation.
- Sidebar navigation on the left, main content on the right, responsive for desktop and mobile.
- Subtle transitions and animations for dark/light mode toggle and page navigation.
